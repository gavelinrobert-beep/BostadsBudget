# Assets Directory

This directory contains image assets for the Bostadsbudget mobile app.

## Generated Assets

All assets feature a blue gradient background (#3B82F6 to #6366F1) with a white house symbol:

- **icon.png** (1024x1024) - Main app icon with rounded corners
- **splash.png** (1284x2778) - Splash screen with house icon and "Bostadsbudget" text
- **adaptive-icon.png** (1024x1024) - Android adaptive icon (transparent foreground)
- **favicon.png** (48x48) - Web favicon
